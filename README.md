# APA Formatting Lab

Interactive APA Title Page and Page Formatting Lab for online courses.

## What this lab covers

- 1-inch margins
- Times New Roman, 12-point font
- Double spacing
- APA title page placement
- Centering title page information
- Page number only in the upper-right corner
- Bold title only
- Extra double-spaced blank line after the title
- APA title case
- Student title page information order
- Transition to Page 2
- First-line paragraph indent
- Optional Level 1 APA heading
